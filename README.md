#### 介绍
SSAdmin 是一个基于 Spring Boot、Mybatis、Spring Security、Mysql、Redis 和 Vue2 的前后端分离的通用后台管理系统。

项目以教学为主要目的，帮助开发者更好地学习和理解当前流行的开发技术和实践。
